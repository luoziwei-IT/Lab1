# Lab1
Active Directory Domain Controller (First Lab)

##Overview
This is a project that demonstrates how to setup and configure a Windows Active Directory domain environment in a virtualized lab. The goal of this lab was to gain hands-on experience with Windows Server, Active Directory, DNS, and basic user management.

##Tools Used
-Hypervisor: VirtualBox
-Server OS: Windows Server 2019 (Evaluation)
-Client OS: Windows 10
-Network Type: Internal Network
-Domain Name: Lab1.com

##Objectives
-Install and configure Windows Server in a virtualized environment
-Promote the server to a Domain Controller
-Configure Active Directory Doain Services
-Configure DNS
-Verify Domain Authentication

##Implementation
-Installed Windows Server 2019 
-Renamed the PC for better clarity
-Promote the server to a Domain Controller
-Installed Active Directory Domain Services (AD DS)
-Create New Users
-Create OUs (Organizational Unit) for Users
